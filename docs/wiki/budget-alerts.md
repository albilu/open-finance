# Budget Alerts

← [Wiki Home](HOME.md)

---

## Overview

Budget Alerts send you a notification when spending against a budget crosses a defined percentage threshold. They work automatically alongside the [Notifications](notifications.md) system so you always know when a budget is at risk.

---

## Default Alert Thresholds

| Threshold | What it means                               |
| --------- | ------------------------------------------- |
| 75 %      | Early warning — spending approaching limit  |
| 90 %      | High-risk warning — budget nearly exhausted |
| 100 %     | Budget exceeded                             |

All three thresholds are active by default when you create a budget.

---

## How Alerts Work

Open-Finance evaluates your budget progress every time a transaction is recorded or imported. If your spending crosses a threshold for the first time in the current period, an alert is created and a notification is sent to your notification centre.

Alerts reset at the start of each new budget period (month, quarter, or year), so you start fresh every cycle.

---

## Configuring Alerts

Go to **Budgets → select a budget → Alert Settings**.

For each budget you can:

- Enable or disable individual thresholds (75%, 90%, 100%)
- Add a custom threshold (e.g., 80%)
