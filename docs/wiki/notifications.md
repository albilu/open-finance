## Overview

The notification centre aggregates in-app alerts from multiple sources — budget thresholds, financial insights, anomaly detection, and system events — into a single, dismissable feed accessible from the bell icon in the top navigation bar.

---

## Notification Sources

| Source            | When you receive a notification                                     |
| ----------------- | ------------------------------------------------------------------- |
| Budget Alert      | Spending crosses a threshold you configured (75%, 90%, 100%)        |
| Financial Insight | A new insight is generated (spending anomaly, savings opportunity…) |
| Anomaly Detection | An unusual transaction is detected by the daily check               |
| System            | App events such as backup completed or import finished              |

---

## Managing Notifications

- **View notifications:** click the bell icon in the navigation bar. A badge shows the number of unread notifications.
- **Mark as read:** open a notification or click **Mark as read** on individual items.
- **Mark all as read:** click **Mark All as Read** at the top of the notification panel.
- **Dismiss:** click the delete icon next to a notification to remove it.

---

## Notification Preferences

You can control which notification types you receive from **Settings → Notifications**. See [Settings & Profile](settings.md) for details.
